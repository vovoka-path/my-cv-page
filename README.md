## :key: Key features
 - Website optimization for search engines (**SEO**)
 - Rich **snippets** for social media using Open Graph
 - Use **schema.org**
 - The site is **adaptive** for any devices and screen sizes
 - **Loading speed** acceleration.
 - **PWA**
